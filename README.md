## 🚀 Installation

```
# clone repository
git clone https://github.com/ivanlori/Vuemmerce.git

# install dependencies
yarn install

# serve with hot reload at localhost:3000
yarn dev

# generate static pages via nuxt.js for static hosting
yarn build
```
